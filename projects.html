BookWorm

Java | Sep - Dec 2020

BookWorm is an Android application that simplifies borrowing and lending books amongst peers. Books are easily added to your profile via ISBN scanning and Requests are managed in a dedicated Requests tab with the ability to set pickup locations.

I played a large role in designing and maintaining the user interface as well as functionality for viewing, editing and adding a book.

Check out the demo video here.

Team Members: Hudson Shykowski, Richmond Naviza, Isaak Coleman, Jawad Hossain, Arden Bautista

Microsoft News for Duo

Xamarin | Jan - Apr 2020

As part of the Microsoft Garage program, my fellow interns and I had the opportunity to reimagine the Microsoft News Android applicaiton for the Surface Duo. We had to make considerations for dual screen interactions and bring them to life using the Duo SDK.

I took on the role of UX Champion and payed careful attention to UX changes in code review. I re-implemented the horizontal navigation to function vertically to better suit the dual-sreen form factor as well as delivered on feature dialogs to educate users on the improved experience.

Read more about the process here.

Team Members: Natalia Maximo, Eishan Lawrence, Rithvik Alluri, Sara Li, Judy Zhong, Eric Sun

Algorithm Visualizer

C++ | Mar - Apr 2019

Algorithm Visualizer is an application which allows the user to construct a graph and visualize a graph search algorithm on it from breadth-first search, depth-first search and Dijkstra's algorithm.

I played a large role in constructing the user interface including sprite placement and logic for connecting nodes as well as translating the user-generated graph into a useable data structure.

Team Members: Samuel Lojpur
